Connection

Implements a database connection on top of PDO.

Signature

class Connection

Methods

Method Arguments Returns Description
connect DataSource $dataSource Opens connection to database server.

Throws ConnectionException: If connection to SQL server fails
keepAlive Restores connection to database server in case it got closed unexpectedly.
disconnect Closes connection to database server.
reconnect Reconnects to database server.
transaction Transaction Operates with transactions on current connection. NOTE: this does not automatically start a transaction. To do that, call begin method.
createStatement Statement Creates a statement on current connection.
createPreparedStatement PreparedStatement Creates a prepared statement on current connection.
getAutoCommit boolean Returns whether or not statements executed on server are commited by default.
setAutoCommit boolean $value Sets whether or not statements executed on server are commited by default.
getConnectionTimeout integer Gets connection timeout from database server. (Not supported by all drivers)
setConnectionTimeout integer $value Sets connection timeout on database server. (Not supported by all drivers)
getPersistent boolean Returns whether or not current connection is persistent.
setPersistent boolean $value Sets whether or not current connection is persistent.

ConnectionFactory

Implements a singleton factory for multiple SQL servers connection.

Signature

class ConnectionFactory

Methods

Method Arguments Returns Description
setDataSource string $serverName,
DataSource $dataSource 		Registers a data source object encapsulatings connection info based on unique server identifier.

Access modifiers: static
getInstance string $serverName Connection Opens connection to database server (if not already open) according to DataSource and returns an object of that connection to delegate operations to.

Access modifiers: static

Throws ConnectionException: If connection to database server fails.
__destruct Disconnects from database server automatically.

PreparedStatement

Implements a database prepared statement on top of PDO.

Signature

class PreparedStatement

Methods

Method Arguments Returns Description
__construct \PDO $PDO Creates a SQL prepared statement object automatically.
prepare string $query Prepares a statement for execution.
bind string $parameter,
mixed $value,
integer $dataType 		Binds a value to a prepared statement.

Throws Exception: If developer tries to bind a parameter to a query that wasn't prepared.
execute string[string] $boundParameters StatementResults Executes a prepared statement.

Throws Exception: If developer tries to execute a query that wasn't prepared.
Throws StatementException: If query execution fails.
